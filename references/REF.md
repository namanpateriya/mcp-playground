# MCP Concepts

## Resources

Read-only information.

Examples:

```text
project://status
project://risks
project://timeline
```

---

## Tools

Executable functionality.

Examples:

```text
summarize_project
calculate_project_health
generate_status_report
```

---

## Discovery

Clients discover:

- resources
- tools

without prior knowledge.

---

# Why MCP Matters

Without MCP:

```text
Model
 ↓
Custom Integration
 ↓
Tool
```

With MCP:

```text
Model
 ↓
MCP Client
 ↓
MCP Server
 ↓
Resources + Tools
```
